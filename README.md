# 🤖 AI Expense Tracker
<img width="1564" height="161" alt="Screenshot 2026-04-10 131306" src="https://github.com/user-attachments/assets/de360d7a-f85d-451b-b5c1-2118aee77483" />
A smart personal finance tool that uses **Gemini 1.5 Flash** to automatically categorize expenses from natural language input.

## 🌟 Features
- **Natural Language Processing:** Just type what you spent, and the AI handles the rest.
- **SQL Backend:** Stores data locally in a SQLite database for privacy.
- **Automated Categorization:** Automatically splits input into 'Food', 'Transport', 'Utilities', etc.

## 🛠️ Tech Stack
- **Language:** Python
- **AI:** Google Gemini API
- **Database:** SQLite3

## 🚀 How to Run
1. Clone the repo.
2. Install requirements: `pip install google-generativeai`
3. Add your Gemini API key to `main.py`.
4. Run `python main.py`.
